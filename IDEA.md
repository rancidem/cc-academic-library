# cc-academic - Personal Academic Tools Library

> A curated collection of Claude Code academic, scientific, and research-oriented skills and tools.

## Project Brief

Create a structured, searchable, and easily maintainable inventory of Claude Code tools, skills, and resources with a focus on academic writing workflows.

## Target Repos

- `academic-paper-skills`
- `wtf-p`
- `MySkills`
- `claude-scientific-writer`
- `scientific-agent-skills`

## Core Goals

- Catalog tools, skills, and resources in a consistent and standardized format
- Enable fast retrieval of information through clear structure and simple search
- Support organization through categories, tagging, and modular entries
- Make workflows reusable by documenting them as discrete, indexed skills

## Design Principles

- Markdown-first and easy to scan
- Flat, predictable directory structure
- Internal links for navigation
- Entry templates that make comparison easier
- Low-maintenance by default

## Working Definition

The library is a markdown-based indexing system for Claude Code-related tools, skills, and supporting resources. Each entry should explain what the item is, what it contains, when to use it, and how it relates to the rest of the library.
