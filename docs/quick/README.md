# Quick Tasks

Use this directory for short-lived task notes, ad hoc follow-ups, and quick-task execution traces.

## Rules

- Keep entries small and disposable.
- Do not store canonical docs here.
- Delete or archive completed quick-task notes when they are no longer useful.

## Suggested Contents

- `TODO.md` for temporary task capture
- `notes.md` for ephemeral context
- `decision.md` for a one-off choice that should inform the next step
