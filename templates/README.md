# Templates

Reusable prompt and document templates live here. The folder is the canonical template library for the repo.

## At A Glance

| Area | Path | Purpose |
|---|---|---|
| Project context | `templates/project-context/` | Focused guidance files for planning and writing |
| Slides | `templates/slides/` | Slide-oriented template variants |
| Posters | `templates/posters/` | Poster-oriented template variants |

## Common Files

- `project.md` - project framing template
- `roadmap.md` - milestone roadmap template
- `state.md` - state tracking template
- `summary.md` - summary template
- `UAT.md` - user acceptance testing template

## Related Surfaces

- `commands/` uses these templates for several planning and writing workflows.
- `skills/` may reference these templates in content-generation workflows.
- `inventory.json` tracks the current template set for drift checks.
