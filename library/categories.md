# Task Categories

Use this page only when you want a backup task-first view rather than the repository-first dashboard.

## Review, Download, Install

| Need | Best route | Why it helps |
|---|---|---|
| Review quickly | [Library inventory](inventory.md) | Fastest way to scan commands, agents, skills, and tools in one place. |
| Compare repos | [Repository summaries](repositories.md) | Best repo-level comparison when you want to choose the right source. |
| Download raw files | [Skills inventory](skills.md) and [Tools](tools.md) | Direct raw file links for skill files, manifests, and package roots. |
| Install or set up | [Repository summaries](repositories.md) and [Resources](resources.md) | Shortest path to install commands, skills, and tools from each source repo. |
| Keep maintenance context | [Maintenance log](../notes/maintenance.md) | Refresh notes and sync history for the library. |
