# Featured Stacks

Curated workflow bundles for the most common ways to use this library after you have already picked a repo or use case.

## Stack Overview

| Stack | Best for | Best first | Install / setup | Download / raw |
|---|---|---|---|---|
| Minimal skill review | Inspecting a single reusable skill pattern quickly | [MySkills](repositories.md#myskills) | Copy `skills/visual-architect/` into your Claude Code skill path | Use the raw `SKILL.md` link in [library/skills.md](skills.md) |
| Paper planning and composition | Planning and writing academic papers with checkpoints | [academic-paper-skills](repositories.md#academic-paper-skills) | Copy `strategist/` and `composer/` into `~/.claude/skills` | Use the raw `SKILL.md` links in [library/skills.md](skills.md) |
| Command-first writing system | Installable academic writing workflows with verification loops | [wtf-p](repositories.md#wtf-p) | Run `npx wtf-p`, or use the global/local install variants | Use raw links in [library/commands.md](commands.md), [agents.md](agents.md), and [tools.md](tools.md) |
| Scientific writing stack | Research-backed writing with plugin, CLI, and package support | [claude-scientific-writer](repositories.md#claude-scientific-writer) | Install the plugin, or use `pip install scientific-writer` / `uv sync` | Use raw links in [library/skills.md](skills.md) and [library/tools.md](tools.md) |
| Large scientific catalog | Broad scientific and research skill coverage | [scientific-agent-skills](repositories.md#scientific-agent-skills) | Run `npx skills add K-Dense-AI/scientific-agent-skills` | Use raw `SKILL.md` links in [library/skills.md](skills.md) |

## How To Use This Page

- Pick the stack that matches your intent first, then open the matching repository summary.
- Treat this page as a helper, not the main dashboard.
- Open the `Best first` link when you want the source repo overview.
- Use the `Install / setup` notes when you want the shortest path to a usable local setup.
- Use the `Download / raw` notes when you want the source files directly.
