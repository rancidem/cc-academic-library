# cc-academic Library

This folder is the working index for the project, but `../LIBRARY.md` is the dashboard you should open first.

## What Lives Here

- [`../LIBRARY.md`](../LIBRARY.md) - master catalog and personal dashboard
- [Repository summaries](repositories.md) - detailed per-repo comparison and entry points
- [Skills inventory](skills.md) - complete skill-file index across cloned repos
- [Tools](tools.md) - reusable environments, wrappers, and helper utilities
- [Resources](resources.md) - supporting references, guides, and maintenance material
- [Entry template](entry-template.md) - shared schema for all entries
- [Tag taxonomy](taxonomy.md) - controlled vocabulary for search and filtering

## How To Use It

1. Start with `../LIBRARY.md` for the current state and best entry points.
2. Use `repositories.md` when you want richer per-repo summaries.
3. Use `skills.md` when you need an exact skill file path.
4. Use `tools.md` and `resources.md` when you want reusable references, not just repo navigation.
5. Tag new entries with the controlled taxonomy before expanding the index.

## Status

The library is organized for fast personal use and ongoing maintenance.
